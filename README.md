# Sadaf Riaz - Personal Dashboard

A modern, responsive personal dashboard showcasing my skills, projects, and professional portfolio as a Computer Science student. Built with pure HTML, CSS, and JavaScript.
# Features

# Design & UI
- **Elegant Pink-Grey Theme**: Custom color scheme with soft pink accents
- **Dark/Light Mode**: Toggle between themes with persistent preferences
- **Responsive Design**: Fully responsive across all devices (mobile, tablet, desktop)
- **Smooth Animations**: CSS transitions and hover effects
- **Glass Morphism**: Modern glass-like card designs

# Interactive Features
- **User Authentication**: Login/Signup system with localStorage
- **Skill Progress Bars**: Animated proficiency indicators
- **Project Filtering**: Filter projects by technology (AI/ML, Web, VR/AR)
- **Session Timer**: Track time spent on the dashboard
- **Contact Form**: Fully validated contact form with error handling
- **Collapsible Sidebar**: Responsive navigation sidebar



#  Pages

#  Home Page (`index.html`)
- Personal introduction and bio
- Profile picture and professional summary
- Theme switcher (light/dark mode)
- Session timer tracking
- Contact form with validation

#  Skills Page (`skill.html`)
- Technical skills categorized by domain
- Animated progress bars showing proficiency levels
- Skills statistics and overview
- Frontend, Backend, and Database skills

#  Projects Page (`project.html`)
- Project portfolio with detailed descriptions
- Filter projects by technology (AI/ML, Web Development, VR/AR)
- Each project includes:
  - Problem statement
  - Unique proposition
  - Technologies used
  - Competitor analysis
  - GitHub links

#  Authentication Pages
- **Login Page** (`login.html`): User authentication
- **Signup Page** (`signup.html`): New user registration

#  Technologies Used

# Frontend
- **HTML5**: Semantic markup and structure
- **CSS3**: Custom properties, Flexbox, Grid, Animations
- **JavaScript (ES6+)**: DOM manipulation, Local Storage, Form validation

# Styling & Design
- **Custom CSS Variables**: Consistent theming system
- **Google Fonts**: Poppins font family
- **CSS Grid & Flexbox**: Modern layout techniques
- **CSS Transitions**: Smooth animations and hover effects
